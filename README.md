# 🏦 API Bancária Assíncrona com FastAPI

Este repositório contém a solução para o desafio de desenvolvimento de um sistema bancário moderno, utilizando **Python** e o framework
**FastAPI**. O projeto aplica conceitos avançados de **Programação Orientada a Objetos (POO)** e processamento assíncrono para gerenciar
clientes, contas e transações.

## 🚀 Tecnologias Utilizadas

* **Python 3.10+**: Linguagem base do projeto.
* **FastAPI**: Framework web de alta performance para construção de APIs.
* **Pydantic**: Validação de dados e definição de schemas.
* **Uvicorn**: Servidor ASGI para execução da aplicação.
* **Docker**: Containerização para facilitar o deploy e a execução.

## 📂 Estrutura de Arquivos

* `app/`: Núcleo da aplicação contendo as rotas (`main.py`), modelos de dados (`schemas.py`) e lógica de persistência em memória (`database.py`).
* `docs_screenshots/`: Documentação visual da API em funcionamento.
* `Dockerfile`: Configuração para criação da imagem Docker do projeto.
* `requirements.txt`: Dependências necessárias para o ambiente.
* `.gitignore`: Configuração para manter o repositório limpo.
